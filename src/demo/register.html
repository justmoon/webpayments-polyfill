Wallet Registration WALLET

A wallet can register itself as the default handler for a payment method.

Web Payments (and Interledger) are about choice. There will be many providers for each payment method. That means a wallet has to register itself:

Example Wallet is not your default demo wallet.

Behind the scenes

In order to trigger the registration as a handler for this payment method, the provider makes a call to navigator.registerPaymentHandler: 

function registerHandler() {
  navigator.registerPaymentHandler('demo', 'https://wallet.example/pay')
}
